# J-sparrow


## Introduction
J-sparrow是由Java编写, 实现了运维4A标准的堡垒机系统, 可部署在Windows或Linux系统中,
后端基于SpringBoot框架, 前端: layui

## Features
- 用户管理&权限分配
- 主机信息管理  
- 通过网页的远程SSH连接
- 审计功能, 记录所有SSH操作日志及任务记录
- 异步批量任务, 文件分发, 可实时查看结果
- 基于Quartz的定时任务, 支持在线配置
